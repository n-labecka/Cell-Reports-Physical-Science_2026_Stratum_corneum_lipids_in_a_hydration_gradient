# Cell-Reports-Physical-Science_2026_Stratum_corneum_lipids_in_a_hydration_gradient

Article: Labecka et al. Stratum corneum lipids in a hydration gradient: Robust lamellar organization and responsive fluidity, Cell Reports Physical Science, 2026.

The GitHub repository contain raw and processed data sets, together with codes which were used to construct the figures in the main article and SI section.

The data and description can be found in the Releases v1.0 in the repository.
